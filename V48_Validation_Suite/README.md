# Dome Cosmology V48 - Empirical Validation Suite

## Repository Purpose

This directory contains code, data, and predictions for the Dome Cosmology model.
All files are timestamped to establish priority of discovery.

### Prediction Frameworks

- `predict_2026_eclipse.py`: 2026 Eclipse prediction generation based on the 1.67 nT/μGal coupling parameter.
- `annual_aberration_model.py`: Aether refractive gradient aberration modeling.
- `parallax_wobble_model.py`: Parallax as firmament wobble.
- `seismic_reflection_search.py`: Framework to hunt for 12,700 km seismic depth reflections.
- `schumann_drift_analysis.py`: Framework to parse the predicted 0.01 Hz/decade SR upward drift.
- `coronal_hole_correlation.py`: Framework to correlate solar wind speed against the acceleration of North Magnetic Pole drift.
- `create_master_predictions.py`: Compiler building the master predictions document from generated JSONs.
